This project provides sandbox for developing custom quizzes at [Stepic](http://stepic.org).


Requirements
===

Python >=3.4 required.

Documentation
===

http://stepic-plugins.readthedocs.org

or

```
pip install -r requirements.txt
cd doc
make html
your-browser _build/html/index.html
```
